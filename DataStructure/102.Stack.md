# 102. Stack

> ### Stack
* **개념**
    - 한 쪽 끝에서만 데이터의 삽입, 삭제가 가능한 후입선출(LIFO) 형식의 자료 구조

* **연산**
    - top(): 스택에서 가장 위에 있는 데이터 반환
    - pop(): 스택에서 가장 위에 있는 데이터 삭제
    - push(data): 데이터를 스택의 가장 윗 부분에 삽입
    - isEmpty(): 스택이 빈 경우 true, 데이터가 하나라도 있으면 false
