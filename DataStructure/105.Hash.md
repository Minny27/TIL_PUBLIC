# 105. Hash

> ### Hash
* **개념**
    - 해시 함수로 얻어진 값을 해시라고 한다.
    - 해시 테이블(hash table), 해시 맵(hash map)는 키(Key)를 값(Value)에 매핑할 수 있는 구조

* **특징**
    - key의 중복을 허용하지 않는다.
    - value는 중복을 허용한다.
    - value 추가, 삭제, 접근의 시간복잡도는 모두 O(1), 단 Collision이 나지 않을 때만
    - 하나의 key에 2개 이상의 값이 충돌하는 해시 충돌(Hash Collision)이 날 수 있음.
