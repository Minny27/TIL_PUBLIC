# 108. Functions

> ### Functions 선언
함수를 정의할 때는 (매개변수: 타입)의 형태로 정의하고  
함수를 호출할 때는 (매개변수: 변수)의 형태로 호출해야한다.
* **반환 없는 경우**
    ```swift
    func a(name: String) {
        print("hello \(name)!")
    }

    a(name: "철수")
    ```
* **Int형 반환**
    ```swift
    func a(name: String) -> String {
        let str = "hello \(name)!"
        return str
    }

    print(a(name: "철수"))
    ```
* **매개변수가 여러개인 경우**
    ```swift
    func sum(a: Int, b: Int) -> Int {
        let tmp = a + b
        return tmp
    }

    print(sum(a: 5, b: 9))
    ```
