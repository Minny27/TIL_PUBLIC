# 102. 출력

> ### 2차원 배열
* **정수, 문자열**
    ```swift
    let nArr : [[Int]] = [[1,2,3], [3,4,5]]
    for i in nArr {
        for j in i {
            print(j, terminator: " ")
        }
        print()
    }
    ```

* **문자열 합쳐서 출력**
    ```swift
    let sArr : [[String]] = [["a","b","b"], ["d","e","f"]]
    for str in sArr {
        print(str.joined())
    }
    ```
