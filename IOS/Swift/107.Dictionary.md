# 107. Dictionary

> ### Dictionary 선언
* **기본 선언**
    ```swift
    var a = [String: Int]()
    var a: [String: Int] = [:]
    var a: [String: String] = ["name": "Lee", "nickName": "kingkong"]
    ```

***

> ### Dictionary 수정 및 꺼내기
* **수정**
    ```swift
    var dict: [String: String] = ["name": "Lee", "nickName": "kingkong"]
    dict["name"] = "Kim"
    
    dict["name"] = nil
    값을 nil로 설정하면 키가 없어진다.
    ```

* **꺼내기**
    ```swift
    let name = dict["name"]
    값이 없을 수도 있으므로 Optional type으로 반환한다.

    if let value = dict["name"] {
        print(value)
    }
    옵셔널 바인딩을 이용하면 고유한 값을 얻을 수 있다.
    ```
